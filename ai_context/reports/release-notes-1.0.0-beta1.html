1.0.0-beta1

129 credited issues

Context Control Center (CCC) beta1 introduces the first full beta of Drupal’s context management layer for AI. Context items are now proper content entities with drafts, revisions, revision comparison, moderation workflows, and scheduling—bringing familiar editorial workflows from nodes to AI context. A new context scope system lets you target context by entity, site section, language, tags, and use case, with subcontext support for required and conditional inclusion. Multilingual support, a markdown editor, usage tracking, an overview dashboard, and improved list filtering help teams author, organize, and monitor context at scale. CCC is also decoupled from AI Agents so context can be managed and delivered independently, with Drupal CMS 2.0 install support and clearer admin navigation throughout. This release hardens the foundation for production use. Architecture and security reviews, expanded automated testing, and significant refactoring improve performance, validation, and maintainability ahead of 1.0. Bug fixes address installation on Drupal 10 and 11, scheduler and moderation conflicts, toolbar and icon regressions, revision UI issues, and scope configuration edge cases. Documentation, project page updates, and demo work round out a milestone aimed at making CCC a dependable core building block for AI-enabled Drupal sites.
